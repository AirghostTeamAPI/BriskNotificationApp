# Brisk Notification App
An app that facilitates standard procedures for drivers in the event of a problem with their vehicle. In the application, he will always have notifications about warnings and recommendations coming directly from the responsible company.

# Instalation and Execute

## Back-end
1. Inside the directory '.\BriskNotificationApp\Back-end\config', create a file 'default.json'
2. Inside the 'default.json' file, add the following keys:
   * "mongoURI": bank's credential (URI)
   * "jwtSecret": PERGUNTAR PARA O HENRIQUE

3. Inside the directory '.\BriskNotificationApp\Back-end\', create a file '.env'

4. Inside the '.env file', add the key 'jwtSecret = PERGUNTAR PRO HENRIQUE' 

5. Open a new terminal and execute this comands in sequence:

    * cd .\Back-end\
    * yarn install
    * yarn server

## Front-end
1. Open a new terminal and execute this comands in sequence:
    * cd .\Front-end\
    * npm install
    
    * npm start
